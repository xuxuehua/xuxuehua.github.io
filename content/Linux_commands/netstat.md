# netstat

netstat 命令显示网络连接、路由表、接口统计、伪装连接和多播连接等信息



## OPTIONS

### -a all 

-a Show all socket information



### -r routing

Show routing information



### -i 接口

Show network interface statistics





### -s 协议

Show network protocol statistics





```
# netstat -tulpn
# netstat -nat
```

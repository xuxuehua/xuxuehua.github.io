# locale 

查看本地语言环境

```
(env3.5.3) xhxu-mac:visibleworld xhxu$ locale
LANG="en_US.UTF-8"
LC_COLLATE="en_US.UTF-8"
LC_CTYPE="en_US.UTF-8"
LC_MESSAGES="en_US.UTF-8"
LC_MONETARY="en_US.UTF-8"
LC_NUMERIC="en_US.UTF-8"
LC_TIME="en_US.UTF-8"
LC_ALL=
```



`export LANG=en_US.UTF-8` 可以更改

for i in range(1):

for vs while

下面二者等价

i = 0
while i < 1000000:
    i += 1

for i in range(0, 1000000):
    pass

要知道，range()函数是直接由C语言写的，调用它速度非常快。而while循环中的“i += 1”这个操作，得通
过Python的解释器间接调用底层的C语言;并且这个简单的操作，又涉及到了对象的创建和删除(因为i是整
型，是immutable，i += 1相当于i = new int(i + 1))。所以，显然，for循环的效率更胜一筹。

break

结束整个循环, 以后的循环都不会被执行

for i in range(5):
    if i == 2:
        break
    print(i)
>>>
0
